\# ALU Regex Data Extraction



\*\*Repository:\*\* alu\_regex-data-extraction-unaiemi-dot  

\*\*Author:\*\* Ulfat Naiemi  



\## Overview



This project is a \*\*regex-based data extraction tool\*\* implemented in JavaScript.  

It processes raw text data and extracts structured information, including:



\- Emails

\- URLs

\- Phone numbers

\- Credit card numbers (masked for security)

\- Times (12h and 24h formats)



The program demonstrates \*\*security awareness\*\*, ignoring malformed or malicious inputs.
